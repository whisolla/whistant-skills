## 📝 Copywriting

### Headline Formula (4 patterns)

**Pattern 1 — Specificity**: `[Number] Ways to Get [Result]`  
→ "27 Ways to Reduce Churn"

**Pattern 2 — Transformation**: `[Adjective] Ways to Get [Result] Without [Pain]`  
→ "Effortless Ways to Get 3x More Leads Without Cold Calling"

**Pattern 3 — Question**: `Is Your [Problem]? Here's How to Fix It.`  
→ "Is Your Landing Page Losing 70% of Visitors? Here's Why."

**Pattern 4 — Pattern Interrupt**: `[Counter-intuitive Claim]`  
→ "Most 'Expert' Landing Pages Are Destroying Conversions"

### CTA Button Copy
| Bad | Better |
|-----|--------|
| "Submit" | "Get My Free Quote" |
| "Click Here" | "See How It Works" |
| "Buy Now" | "Start My 14-Day Trial" |

### Email Subject Lines
- Curiosity: "I saw your post on [topic]..."
- Specificity: "3 things most [profession] miss about [topic]"
- Social proof: "[Famous person] uses this trick for [result]"
- Urgency: "Last chance: [offer] expires [date]"

## 📈 CRO (Conversion Rate Optimization)

### Landing Page Audit Checklist
1. **Hero**: Clear headline matching search intent + subheadline expanding
2. **Subheadline**: Elaborates on the transformation
3. **CTA**: Above fold, above the fold, above the fold
4. **Social proof**: Testimonial or logo strip within 2 scrolls
5. **Pain → Solution**: "We know you [pain]. Here's how we [solution]."
6. **Features → Benefits**: Don't describe features, describe what they unlock
7. **Visual hierarchy**: 1 main CTA per page

### A/B Test Setup
```
Hypothesis: "If we [change X], then [metric Y] will [increase/decrease]
  by [Z]% because [rationale]."

Control: [current version]
Test: [change being tested]
Segment: [who this applies to]
Metric: [primary: conversion rate | secondary: bounce rate]
Min sample: [use online calculator — typically 1,000+ per variation]
Duration: [minimum 2 weeks, or until statistical significance]
```

## 🔍 SEO

### SEO Audit Checklist
1. Title tag: `<title>[Primary Keyword] — [Brand]` (50-60 chars)
2. Meta description: 150-160 chars, includes keyword + CTA
3. H1: Exactly one, contains primary keyword
4. H2-H6: Logical hierarchy, keyword-rich
5. Internal links: every page links to 3+ related pages
6. Images: alt text on all meaningful images
7. Page speed: <3s load on mobile
8. Core Web Vitals: LCP <2.5s, CLS <0.1, INP <200ms
9. Schema markup: Organization, Article, FAQ, Product as applicable
10. Canonical URL: self-referencing on all pages

### Programmatic SEO Template
```
/[keyword-plural]-[modifier].html

Template structure:
- H1: [Keyword] [Modifier] — [Pain/Goal]
- Intro: 100 words, keyword in first 50
- [N] Subheadings: [How to] / [Best] / [X vs Y] / [What is]
- FAQ section: 5-8 questions from "People Also Ask"
- CTA: links to main product/service page
```

## 📧 Email Sequences

### Welcome Sequence (Day 0 → 7)
| Day | Goal | Content |
|-----|------|---------|
| Day 0 | Deliver + set expectations | What they signed up for + what comes next |
| Day 1 | Prove value | Best content/resource (lead magnet follow-through) |
| Day 3 | Share origin story | Why this was built, who it's for |
| Day 5 | Social proof | 2-3 testimonials + context |
| Day 7 | Soft ask | Survey question or reply-asking email |

### Drip Campaign Triggers
- New subscriber → welcome sequence
- Free trial → trial activation sequence
- Abandoned cart → 3-email sequence (1h, 24h, 72h)
- Purchase → post-purchase sequence (receipt, onboarding, upsell)
- Lapsed user → re-engagement sequence (win-back offer)

## 💰 Pricing Strategy

### Value-Based Pricing Framework
```
Price = [Value Delivered in $] × [Capture Rate]

Example: Tool saves 10 hrs/week at $100/hr = $1,000/week value
Capture 10% = $100/month price point
```

### Packaging Tiers (3-tier strategy)
```
Tier 1 — Entry:      [Minimum useful feature]  + [No risk CTA]
Tier 2 — Popular:     [Full feature set]        + [Best value CTA] ← highlight this
Tier 3 — Enterprise:  [Everything + white glove + custom] + [Contact sales]
```

### Churn Reduction Triggers
- Day 1: Onboarding email with clear next step
- Day 7: "How's it going?" + offer help
- Day 14: Feature highlight they haven't used
- Day 30: "Results so far?" survey
- Day 60+: Proactive check-in for power users

## 📊 Analytics

### GA4 Key Events to Track
- `sign_up` — user completes registration
- `trial_start` — user starts free trial
- `purchase` — transaction completed
- `page_view` — key landing pages
- `video_start` — engagement signal
- `content_download` — high-intent action

### Funnel to Monitor
```
Awareness → Landing Page (measure bounce rate)
  → Sign Up Flow (measure drop-off at each step)
    → Activation (first meaningful action)
      → Retention (Day 1, 7, 30 retention cohorts)
        → Revenue (MRR, ARPU, LTV)
```

## 🧠 Marketing Psychology

### Cognitive Biases to Use
- **Scarcity**: "Only 3 spots left" (only if true)
- **Social proof**: "[X companies] already using this"
- **Authority**: "Featured in [publication]" or "Used by [expert]"
- **Reciprocity**: Free tool, guide, or template → increases conversion
- **Loss aversion**: "Don't lose access to [feature]" > "Gain access"
- **Anchoring**: Show high price first → medium price looks cheap

### Persuasion Checklist
1. Does the headline state the transformation or outcome?
2. Is the specific number/category mentioned (not generic)?
3. Is the offer low-risk (trial, guarantee, cancel anytime)?
4. Is the next step obvious and singular?
